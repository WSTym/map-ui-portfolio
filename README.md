# Premium Map UI - Fuel Stations Portfolio

This project is a high-performance, premium Map UI built as a portfolio piece. It demonstrates a responsive, map-first user experience for navigating a large dataset of fuel stations, specifically designed to meet the requirements of modern web applications.

## 🚀 Live Demo
*(You can host this static build on GitHub Pages or Cloudflare Pages and place the link here)*

## 🎯 Key Features

- **Map-First Experience:** 100vh full-screen Leaflet map using premium Dark Theme tiles (CartoDB Dark Matter).
- **Glassmorphism UI:** A sleek, frosted-glass right-hand sidebar for desktop users that seamlessly transforms into an interactive **Bottom Sheet** on mobile devices.
- **Marker Clustering:** Efficient rendering of 150+ station markers using `Leaflet.markercluster`, ensuring smooth performance and a decluttered map.
- **Map ↔️ List Sync:**
  - **Bounds-based Filtering ("This Area"):** As you pan and zoom the map, the list automatically updates to show only the stations currently visible on your screen.
  - **Bi-directional Focus:** Clicking a station in the list flies the map to that specific marker. Clicking a marker on the map scrolls the list to the corresponding card.
- **Geolocation ("Near Me"):** Prompts for user location and dynamically sorts the station list by distance, immediately flying the map to the user's current location.
- **Real-time Filtering:** Instantly filter stations by status (e.g., "Open Now") or availability of pricing data.

## 🛠️ Technology Stack

- **Vite:** Blazing fast frontend build tool.
- **TypeScript & Vanilla JavaScript:** No heavy frameworks (like React or Vue), ensuring maximum performance and aligning perfectly with Cloudflare Worker HTML/JS serving constraints.
- **Leaflet & Leaflet.markercluster:** Core mapping library and clustering plugin.
- **CSS Custom Properties (Vanilla CSS):** A robust design system using variables for easy theming, maintaining a strict and highly polished dark aesthetic.

## 📦 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/WSTym/map-ui-portfolio.git
   cd map-ui-portfolio
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

4. **Build for production:**
   To generate the static assets (which can be served directly from a Cloudflare Worker or GitHub Pages):
   ```bash
   npm run build
   ```
   The compiled assets will be in the `/dist` directory.

## 🗄️ Data Structure (Mock API)

The application fetches data from a static `stations.json` file located in the `/public` directory, simulating a `GET /api/stations` endpoint. This data is generated by `generate_mocks.py` and includes fields such as `name`, `area`, `status`, `fuel`, `queue`, `price`, and spatial coordinates (`lat`, `lng`).

## 📱 Responsive Details

- **Desktop ( > 768px ):** An absolute positioned sidebar on the right side with its internal scroll, overlaying the map map seamlessly.
- **Mobile ( <= 768px ):** The sidebar converts to a native-feeling Bottom Sheet with a subtle grab handle, minimizing map obstruction while remaining highly accessible.
